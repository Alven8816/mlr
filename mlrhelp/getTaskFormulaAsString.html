Get formula of a task as a string.

Usage

getTaskFormulaAsString(x)

Arguments

x
[SupervisedTask | TaskDesc] Task or its description object.

Value

[character(1)].

Description

This is simply “ ~ .”.

Examples

task <- makeClassifTask(data = iris, target = "Species")
getTaskFormulaAsString(task)
