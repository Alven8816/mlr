Induced model of learner.

Usage

makeWrappedModel(learner, model, task.desc, subset, features, time)

Arguments

learner
[Learner] The learner.
model
[any] Underlying model.
task.desc
[TaskDesc] Task description object.
subset
[integer] Subset used for training.
features
[character] Features used for training.
time
[numeric(1)] Computation time for model fit in seconds.

Value

[WrappedModel].

Description

Result from train. It internally stores the underlying fitted model, the subset used for training, features used for training and computtation time for training.

Details

The constructer makeWrappedModel is only for internal use.
